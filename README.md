# Backend


## Configuration
Create a .env file and put your credentials
### `PORT=<YOUR-PORT>`
### `USER_DB=<YOUR-USER_UD>`
### `HOST=<YOUR-HOST>`
### `DATABASE=<YOUR-DATABASE>`
### `PASSWORD=<YOUR-PASSWORD>`

## Available Scripts

Install Dependencies
### `npm install`

Run
### `npm start`

Dev mode
### `npm run dev`


## App
The apps run on PORT that you declare on the .env file
